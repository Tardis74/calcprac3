#Решение систем линейных алгебраических уравнений. Метод Гаусса

##Компиляция и тестовые данные
Для компиляции основной программы запустить
```bash
make all
```

Для компиляции программы для генерации тестовых данных запустить
```bash
make data_{random|hilbert}
```
Первая программа генерирует матрицу с абсолютно случайными знаениями элементов (да, долго, но абсолютно случайно), вторая программа генерирыет матрицу A как матрицу Гильберта, а матрицу B случайной.

**ВНИМАНИЕ** При запуске программы на сгенерированных данных первым генератором не забудте очекь сильно понадеяться на свою удачу, потому что определитель матрицы может в очень редких случаях оказаться нулевым.

##Запуск программы

Для запруска программы использовать комманду:
```bash
./solver {n}
```
где в качестве n указывается метод (Гаусс без ведущего элемента 0, Гаусс с ведущим элементом 1, Жордан 2). Либо запустить команду
```bash
./solver
```
после чего выслушать его ругательства и понять, что он оидает при запуске.
